# ``Libgnss_Coding``

A high-performance Swift implementation of GNSS-related coding and parity check algorithms.

## Overview

`Libgnss_Coding` provides essential algorithms for processing GNSS (Global Navigation Satellite System) data streams. It includes various Cyclic Redundancy Check (CRC) implementations and Hamming (32, 26) parity algorithms as specified in international standards like IS-GPS-200.
